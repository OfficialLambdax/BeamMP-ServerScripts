# AutoRestart

### What does it do?
Will exit the game server at your desired interval to allow the container the server is running in to restart it.

### Installation
- Put the entire `AutoRestart` folder from `Server/` into your servers `Resources/Server` folder
- Open the `Resources/Server/AutoRestart/main.lua` file in a text editor
- Edit the settings to your liking
- Save the file
- Run the Server and have fun (:
